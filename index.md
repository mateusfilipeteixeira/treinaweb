## Como usar esta trilha

Este site foi montado para pessoas que estão começando do absoluto zero. A ordem dos módulos foi pensada para reduzir ansiedade, construir base forte e evitar o salto prematuro para temas avançados sem fundamento.

Siga a sequência sugerida, faça os exercícios, termine os projetos de cada etapa e trate revisão como parte normal do processo.

## O que você vai desenvolver aqui

- autonomia com computador, arquivos, navegador e segurança básica
- raciocínio lógico para resolver problemas
- entendimento de HTML, CSS e JavaScript
- prática com Git, GitHub e publicação
- pelo menos um projeto final para portfólio

## Para quem este material funciona melhor

- pessoas que nunca programaram
- pessoas que já tentaram aprender e travaram na base
- pessoas que querem uma trilha mais organizada e didática

## Regra de ouro

Avançar rápido não é a meta. Entender e conseguir aplicar é a meta.
